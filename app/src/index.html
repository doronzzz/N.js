Getting started

Start by adding this script to the bottom of you HTML <body> 

				<script src="https://rawgit.com/doronzzz/N.js/master/app/dist/bin.js" type="text/javascript"> 
				</script>

To set your Nurego API key 

				
				<nurego-api-key key="000-000-000-000">
				</nurego-api-key>

To overwrite the Nurego widgets base url use this directive: 

				
				<nurego-js-baseurl url="http://localhost:31001/">
				</nurego-js-baseurl>

To overwrite the Nurego API Server base url (staging/prod/local) use this directive: 

				
				<nurego-api-baseurl url="http://localhost:31001/">
				</nurego-api-baseurl>
